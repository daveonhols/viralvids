Viral Videos

What is this?

What

A proof of concept / portfolio site, tracking Youtube video mentions on Twitter. Stay up to date on the most viral Youtube videos on Twitter in a variety of categories as we track them in real time.

How

We run a node.js back end using the Twitter API to listen for tweets that mention Youtube videos across a range of categories of interest. Each mention is counted, aggregated and sorted in real time and stored in a Mongo DB collection. This express driven front end serves up the underlying data for users to see the videos

Who

David Smith, an experienced back end and data driven developer.
